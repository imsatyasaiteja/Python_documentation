Introduction

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

The uses of python are

  • Web development
  • Software development
  • Mathematics
  • System Scripting

Quick Start

Python is an interpreted programming language. You can write python (.py) file in a text editor and put that file in a python interpreter to execute it. Write the following code in your first python file, you can name it as helloworld.py

Example :

print("Hello, World!")

Now, open your command line, navigate to your directory where you save the python file and run it.

Sometimes, to check a short amount of code we can use the command line itself. Try typing "python" or "py" on the command line to ensure if python is working. Then you can type your code.

Data Types

First, we need to learn about variables. Variables are those which can store values (of specified data type) and can varied at any point of time.

Now, Data types are those that specify the type of a data. Python has the following data-types built-in by default

  • Text type: str
  • Numeric type: int, float, complex
  • sequence type: list, tuple, range
  • boolean type: bool
  • binary type: bytes, bytearray, memoryview
  • set types: set, frozen set

In the above mentioned list, the ones that are on the right side of the hyphen are data types. You can use type() function to obtain the datatype.

Strings

Strings in python are surrounded by either single quotation marks, or double quotation marks.

'hello' is the same as "hello".

You can display a string literal with the print() function

Example :

print("Hello")print('Hello')

Both the lines above give the same result Hello
